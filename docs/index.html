Transfiere archivos entre macOS y dispositivos Android vía USB con una interfaz nativa rápida, segura y moderna.

Características

Explorador Dual

Navega archivos locales y remotos en paneles paralelos con vista de lista o cuadrícula.

Push / Pull

Copia archivos en ambas direcciones con barra de progreso en tiempo real y hasta 5 transferencias concurrentes.

Pausa / Reanudar

Pausa y reanuda transferencias individualmente. Al reanudar retoma desde donde se quedó.

Cancelar

Cancela transferencias en curso. Los archivos parciales se eliminan automáticamente.

Operaciones de Archivos

Crea, renombra y elimina carpetas y archivos en ambos lados con confirmación.

Selección Múltiple

Selecciona varios archivos con checkboxes o atajos de teclado para copiar o eliminar.

Rutas Editables

Navegación por breadcrumbs o escritura manual de ruta. Botón para ir al directorio padre.

Espacio en Disco

Muestra almacenamiento disponible local y remotamente con barras de uso.

Tema Claro / Oscuro

Cambia entre tema oscuro, claro o sigue la preferencia del sistema.

Guía de Uso

1

Conecta tu dispositivo

Conecta tu Android por USB y activa la depuración USB en Ajustes → Opciones de desarrollador. La app detectará el dispositivo automáticamente.

2

Navega y selecciona

Explora los archivos en ambos paneles. Usa los checkboxes o Cmd+clic para seleccionar múltiples archivos. Cambia a vista cuadrícula con el ícono de cuadrícula en el header.

3

Transfiere archivos

Presiona el botón → para copiar a Android (push) o ← para copiar a Mac (pull). Las transferencias aparecen en el panel de progreso en la parte inferior.

4

Gestiona transferencias

Haz hover sobre una transferencia para ver los botones de pausa y cancelar. Al reanudar, la transferencia continúa desde donde se pausó. Hasta 5 archivos se transfieren simultáneamente.

Requisitos

macOS

  • macOS 11 Big Sur o superior
  • Intel o Apple Silicon (M1–M4)

Android

  • Android 5.0 (Lollipop) o superior
  • Depuración USB activada
  • Cable USB para datos

Descarga

  • App nativa construida con Tauri
  • No requiere Java ni Electron
  • Binario firmado para macOS
